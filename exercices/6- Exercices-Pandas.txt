Exercice 1 : L'Aventure des Animaux
Objectif : Explorer les fonctions de base de Pandas (création, sélection, filtrage).
Données : Créez un DataFrame animaux contenant les colonnes Nom, Espèce, Age et Poids. 
Ajoutez des données pour différents animaux (par exemple, un lion, un perroquet, un poisson, etc.).
Tâches :
1. Sélectionnez tous les animaux qui pèsent plus de 10 kg.
2. Trouvez l'animal le plus âgé.
3. Ajoutez une nouvelle colonne Longévité où vous estimez la durée de vie maximale de chaque animal.
4. Créez un nouveau DataFrame où vous ne gardez que les mammifères.

Exercice 2 : La Course des Super-Héros
Objectif : Utiliser des opérations mathématiques et de groupement.
Données : Créez un DataFrame heros avec les colonnes Nom, Vitesse, Force, Intelligence.
Tâches :
1. Ajoutez des super-héros avec leurs attributs (ex : Superman, Iron Man, etc.).
2. Calculez un score total pour chaque héros en additionnant toutes leurs statistiques.
3. Trouvez le héros le plus fort et le plus intelligent.
4. Groupez les héros par score total et affichez les moyennes pour chaque groupe.

Exercice 3 : Le Mystère des Fruits
Objectif : Pratiquer le slicing, le filtrage, et la manipulation de chaînes.
Données : Créez un DataFrame fruits avec les colonnes Nom, Couleur, Prix.
Tâches :
1. Ajoutez une variété de fruits avec leurs informations.
2. Sélectionnez tous les fruits de couleur rouge.
3. Augmentez le prix de tous les fruits de 20%.
4. Ajoutez une colonne Nom_Majuscule qui contient les noms des fruits en majuscules.

Exercice 4 : Le Festival de Musique
Objectif : Fusionner des DataFrames et gérer des données manquantes.
Données : Créez deux DataFrames, artistes avec les colonnes Nom, Genre et Année, et concerts 
avec les colonnes Artiste, Lieu, Date.
Tâches :
1. Ajoutez des artistes et leurs informations dans artistes.
2. Ajoutez des concerts dans concerts, en faisant correspondre certains artistes.
3. Fusionnez les deux DataFrames pour avoir une vue complète des concerts avec les détails des artistes.
4. Remplacez toutes les données manquantes par des valeurs par défaut (par exemple, "Inconnu" pour le lieu).
